                        <h1>
                          How to send receipts to customers via SMS from
                          Loyverse POS
                        </h1>
                        <p>
                          Some Loyverse POS users want to send the receipts to
                          customers' phones via SMS.&nbsp;<br /><br />It is
                          possible to organize sending SMS with the receipts to
                          users from the customer base of the
                          <a href="https://loyverse.com/loyalty-program"
                            >loyalty program</a
                          >.&nbsp;<br />In this tutorial, we will explain how to
                          make such
                          <a href="https://loyverse.com/marketplace"
                            >integration</a
                          >
                          by using
                          <a href="https://loyverse.com/marketplace/integromat"
                            >Make</a
                          >, connected to Loyverse through API, and to SMS
                          sending services such as Twilio and SMS
                          Alert.&nbsp;<br /><br />Before starting organizing
                          such connections, you need to have:
                          <a href="https://loyverse.com/en/signup"
                            >Loyverse account</a
                          >. Also, you need to have a
                          <a href="https://www.make.com/en">Make</a> account and
                          SMS sending service account.
                        </p>
                        <h2>
                          Sending receipt via SMS from Loyverse POS through SMS
                          Alert
                        </h2>
                        <p>
                          <a
                            href="https://www.smsalert.co.in/"
                            rel="noopener external nofollow"
                            target="_blank"
                            >SMS Alert</a
                          >
                          is an SMS-sending service. You need to have an account
                          in SMS Alert, if you do not have - please, create one.
                        </p>
                        <p>
                          Login into Make and select the “<a
                            href="https://www.make.com/en/templates/7952-send-loyverse-receipts-by-sms-using-sms-alert"
                            >Send Loyverse receipts by SMS with SMS Alert</a
                          >” template. You can also send just brief details of
                          the receipt. For this use “<a
                            href="https://www.make.com/en/templates/8077-send-short-loyverse-receipt-by-sms-using-sms-alert?fromImt=1&amp;"
                            >Send short Loyverse receipt by sms using SMS
                            Alert</a
                          >”.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/01-templates.png"
                            alt="Loyverse receipt by sms using SMS Alert"
                            width="1573"
                            height="696"
                          />
                        </p>
                        <p>
                          You will have the option to either “<strong
                            >Start a guided set up</strong
                          >” or “<strong
                            >Create new scenario from template</strong
                          >”. <strong>Start a guide set up step</strong> by step
                          opens each action in order so that you can quickly
                          place in the necessary fields. But for the sake of
                          this guide, we will be using the “<strong
                            >Create new scenario from template</strong
                          >”.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/02-new-scenario.png"
                            alt="Create new scenario from template"
                            width="1920"
                            height="953"
                          />
                        </p>
                        <p>
                          Step 1, you need to add a connection to your Loyverse
                          account.&nbsp;<br />Press the “My Loyverse API Key
                          Connection” dropdown button and select “My Loyverse
                          OAuth 2 connection”. Then, press the “Add” button. It
                          will open a new tab for you that directs to the
                          Loyverse Login page, enter your login credentials and
                          click “Allow”.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/03-credentials.png"
                            alt="enter your login credentials"
                            width="861"
                            height="663"
                          />
                        </p>
                        <p>
                          Enter other details. As "Store ID", you can select a
                          particular store of your Loyverse account. The “Limit”
                          field defines the number of receipts that will be sent
                          via SMS per each scenario execution. For example, if
                          you have up to 100 receipts per hour and run the
                          scenario every 15 minutes, you can set the limit as
                          25.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/04-store-id.png"
                            alt="Store ID"
                            width="948"
                            height="700"
                          />
                        </p>
                        <p>
                          Choose from where you want to start your automation:
                          “from now on or other custom periods”.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/05-choose-start.png"
                            alt="Choose from where to start"
                            width="427"
                            height="276"
                          />
                        </p>
                        <p>
                          <strong>Note:</strong> On Step 2, Step 3 and Step 4
                          you need to choose the same connection as added on
                          Step 1!
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/06-choose-connection.png"
                            alt="Choose the same connection"
                            width="993"
                            height="620"
                          />
                        </p>
                        <p>
                          To establish the connection with SMS Alert, on the
                          <strong>Step 5</strong>, you need to copy your
                          <strong>Username</strong> and
                          <strong>Password</strong> from your SMS Alert account.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/07-username.png"
                            alt="enter your Username and Password"
                            width="323"
                            height="323"
                          />
                        </p>
                        <p>
                          And paste it in the “Create a connection” form in the
                          “User Name” and “Password” fields respectively.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/08-create-connection.png"
                            alt="Create a connection"
                            width="750"
                            height="345"
                          />
                        </p>
                        <p>
                          For testing your template, please, make a testing sale
                          in your POS from the account you are connected with
                          and run your scenario once by clicking the “Run once”
                          button.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/09-run-once.png"
                            alt="run your scenario once by clicking the Run once button"
                            width="931"
                            height="681"
                          />
                        </p>
                        <p>
                          To start sending SMS automatically, switch the
                          Scheduling parameter to “On”.&nbsp;<br /><br />Don’t
                          forget to save your scenario by pressing the “disc”
                          icon.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/10-on-sms-auto.png"
                            alt="send SMS automatically"
                            width="1145"
                            height="653"
                          />
                        </p>
                        <p>
                          After completing all previous steps, your automation
                          is ready to go, and the whole scenario will work
                          automatically! Make will get all necessary information
                          from the Loyverse receipt and pass it to SMS Alert.
                          SMS Alert will send the receipt via SMS to your
                          customer, using the customer’s form’s phone number.
                        </p>
                        <p>
                          During the selling process, assign the ticket to a new
                          or an existing customer with their Phone number.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/11-customer.png"
                            alt="assign the ticket to customer"
                            width="616"
                            height="415"
                          />
                        </p>
                        <p>
                          <strong>Note:</strong> You should use the
                          <strong>international format phone number</strong> in
                          the customer's card, which starts from + and country
                          code.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/12-customers-form.png"
                            alt="customer phone number"
                            width="492"
                            height="492"
                          />
                        </p>
                        <p>
                          To finalize the selling process, click the “New sale”
                          button.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/13-new-sale.png"
                            alt="selling process. New sale button"
                            width="750"
                            height="544"
                          />
                        </p>
                        <p>
                          After this, receipt details will be sent by SMS to the
                          customer’s Phone number.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/14-receipt-details.png"
                            alt="receipt details will be sent to the customers phone number"
                            width="331"
                            height="492"
                          />
                        </p>
                        <p><strong>Note:</strong></p>
                        <p>
                          If you will use “Send short Loyverse receipt by sms
                          using SMS Alert”, customers will get only information
                          about Shops’ name, Receipt number, Total purchase
                          value, information about Points and receipt date
                          instead of all details of the receipt.
                        </p>
                        <p>
                          <img
                            class="img-in-blog"
                            src="/sites/all/themes/loyversecom/images/blog/sms-alert/15-sms-alert.png"
                            alt="Send short Loyverse receipt by sms using SMS Alert"
                            width="202"
                            height="145"
                          />
                        </p>
                        <p>Enjoy using Loyverse integrations!</p>
                        <p>&nbsp;</p>
