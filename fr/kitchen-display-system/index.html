Foire aux Questions

Qu'est-ce que Vemli KDS ?

Vemli KDS est l'application du système d'affichage cuisine qui fonctionne sur les tablettes iPad ou Android et affiche les commandes effectuées depuis le PDV de Vemli. KDS est généralement placé dans la cuisine pour que les cuisiniers puissent voir les informations des commandes et préparer les plats. Laffichage cuisine est un moyen moderne pour rempacer les imprimantes de cuisine traditionelle qui imprime les tickets sur papier. C'est un excellent moyen d'économiser du papier en rendant le processus numérique.

KDS affiche toutes les informations nécessaires sur les commandes cuisine : noms des articles, quantités, modifications, commentaires sur les articles et les tickets ; il indique la couleur des tickets, en fonction du temps écoulé depuis leur création.
