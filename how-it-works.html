Workflow

From table selection to payment — see how Vemli POS streamlines your entire restaurant operation, step by step.

1
🪑 Create Table or Select Existing

Start by creating a new table with guest information or select an existing table. Perfect for managing walk-ins and reservations.

✓ Guest count tracking ✓ Reservation management ✓ Table status tracking
2
📋 Browse Menu & Select Items

Navigate through organised categories (Drinks, Food, Combos) and select items. Touch-friendly interface for fast ordering.

✓ Category navigation ✓ Search functionality ✓ Favourite items
3
🛒 Add to Cart / Basket

Click to add selected items. Real-time updates show current order total and item count.

✓ Real-time total ✓ Quantity adjustment ✓ Cart preview
4
🍳 Select Sides & Extras Restaurant Edition

Pop-up options appear for food items to select sides, cooking preferences, and extras. Customise orders exactly as customers request.

✓ Side selection ✓ Cooking instructions ✓ Extra toppings ✓ Special requests
5
📤 Place Order

Finalise the order and send it to the kitchen instantly. Orders appear immediately on kitchen displays.

✓ Instant kitchen notification ✓ Order confirmation ✓ Order number generation
6
💳 Select Payment Method

Choose from multiple payment options or integrate with Property Management System to charge to hotel rooms.

✓ Cash ✓ Card ✓ Mobile money ✓ PMS integration
7
% Apply Discounts

Apply full or partial discounts before payment. Flexible discount system for promotions, loyalty, or special occasions.

✓ Percentage discounts ✓ Fixed amount ✓ Item-level discounts
8
🧾 Settle Bill

Complete the payment. System auto-calculates change and updates inventory in real-time.

✓ Auto change calculation ✓ Split payments ✓ Tip management ✓ Inventory update
9
🖨️ Print Receipt

Generate and print detailed receipts including item breakdown, taxes, discounts, and payment details.

✓ Itemised receipt ✓ Tax breakdown ✓ Customer copy
10
🍽️ Kitchen Automation Restaurant Edition

Automatic Kitchen Order Ticket (KOT) printing and push to Kitchen Display Screen (KDS) for real-time order tracking.

✓ KOT auto-printing ✓ KDS integration ✓ Order status tracking ✓ Prep time tracking
